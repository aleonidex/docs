Включите для ВМ авторизацию по {{ oslogin }} при подключении к серийной консоли, указав имя ВМ:

```bash
yc compute instance update \
  --name <имя_ВМ> \
  --serial-port-settings ssh-authorization=OS_LOGIN
```
